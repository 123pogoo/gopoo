# GOPOO Facebook 广告投放完整指南

## 🚀 快速开始（5分钟）

### 第一步：创建 Facebook 广告账户（如果还没有）
1. 访问 https://business.facebook.com/
2. 点击"创建账户"
3. 填写基本信息（公司名称、邮箱、电话）
4. 验证邮箱和电话

### 第二步：获取 Facebook Pixel ID
1. 登录 Facebook 商业管理平台
2. 进入"事件管理器"
3. 选择"Web"
4. 创建新的数据源或选择现有的
5. 复制您的 **Pixel ID**（格式：1234567890）

### 第三步：更新网站上的 Pixel ID
在 `index.html` 文件中，找到这一行：
```javascript
fbq('init', '1234567890'); // 请替换为您的 Pixel ID
```
将 `1234567890` 替换为您的实际 Pixel ID。

同时更新 noscript 标签中的 ID：
```html
<noscript><img height="1" width="1" style="display:none" src="https://www.facebook.com/tr?id=YOUR_PIXEL_ID&ev=PageView&noscript=1" /></noscript>
```

### 第四步：验证 Pixel 安装
1. 访问您的网站 pogoo.shop
2. 返回 Facebook 事件管理器
3. 点击"发送测试流量"
4. 应该看到"测试事件已收到"的确认

## 🎯 监测和优化

### 每天检查的关键指标

1. **点击率（CTR）**
   - 目标：> 1.5%
   - 如果 < 1%：调整文案或图片

2. **转化率**
   - 目标：> 2-3%
   - 如果 < 1%：检查网站转化流程

3. **每次点击成本（CPC）**
   - 目标：< NT$100
   - 如果 > NT$150：调整受众或竞价

4. **广告支出回报率（ROAS）**
   - 目标：> 3:1
   - 如果 < 2:1：停止该广告，优化其他

### 优化建议

**如果某个广告表现不佳：**
1. 检查文案是否清晰
2. 检查图片是否吸引人
3. 检查受众定位是否准确
4. 考虑调整出价策略
5. 如果仍无改善，暂停该广告

**如果某个广告表现很好：**
1. 增加该广告的预算
2. 测试相似的受众
3. 尝试扩展到其他地区
4. 创建相似的广告变体

## 💡 常见问题解答

### Q1：为什么我的广告没有被批准？
**A**：Facebook 有严格的广告政策。常见原因：
- 文案中有虚假声明
- 图片质量不够好
- 目标网址有问题
- 包含禁止的词汇

**解决方案**：检查 Facebook 广告政策，修改后重新提交。

### Q2：为什么我的转化率很低？
**A**：可能的原因：
- 网站加载速度慢
- 结账流程复杂
- 产品价格过高
- 受众定位不准确

**解决方案**：优化网站性能，简化结账流程，重新定位受众。

### Q3：我应该投放多久才能看到结果？
**A**：Facebook 需要时间学习和优化。建议：
- 前3天：数据收集阶段，可能效果不佳
- 第4-7天：优化阶段，效果逐渐改善
- 第2周及以后：稳定阶段，效果相对稳定

### Q4：如何计算广告的投资回报率（ROI）？
**A**：
```
ROI = (销售额 - 广告成本) / 广告成本 × 100%

例如：
销售额 = NT$18,000
广告成本 = NT$6,000
ROI = (18,000 - 6,000) / 6,000 × 100% = 200%
```

### Q5：我应该如何分配预算？
**A**：建议按以下比例：
- 功能导向：33%（最高转化）
- 价值导向：25%（强调优惠）
- 社证导向：17%（建立信任）
- 创新导向：17%（差异化）
- 再营销：8%（转化已有访客）
